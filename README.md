# Магазин видеоигр

Аккуратный demo-проект интернет-магазина видеоигр с backend-прокси для AI-запросов.

## Что есть
- современный UI на отдельном HTML/CSS/JS
- каталог с поиском, фильтрами и сортировкой
- подробная карточка игры в модальном окне
- демо-профили пользователей
- отзывы
- AI-помощник для 4 сценариев:
  - AI-консультант
  - интеллектуальный поиск
  - рекомендации
  - сводка отзывов

## Быстрый запуск
```bash
pip install -r requirements.txt
export ARTEMOX_API_KEY="ваш_ключ"
python app.py
```

Откройте: `http://127.0.0.1:8080`

## Termux
```bash
pkg update -y
pkg install -y python
pip install -r requirements.txt
export ARTEMOX_API_KEY="ваш_ключ"
python app.py
```

## Важно
API-ключ лучше передавать через переменные окружения, а не вшивать в код.
