# PlayGrid AI — сборка в EXE без ручной установки библиотек у преподавателя

## Что это
Готовый проект для сборки в Windows `.exe` через PyInstaller.
После сборки преподавателю не нужно ставить Python, Flask или requests — он запускает один файл `PlayGridAI.exe`.

## Что нужно только вам для сборки
- Windows 10/11
- Python 3.11+ или 3.12+
- интернет для первой установки зависимостей

## Как собрать EXE
1. Распакуйте проект.
2. Скопируйте `.env.example` в `.env`.
3. Впишите ваш `ARTEMOX_API_KEY` в `.env`.
4. Запустите `build_windows.bat`.
5. Готовый файл будет здесь:
   `dist\PlayGridAI.exe`

## Как запускать потом
1. Положите рядом с `PlayGridAI.exe` файл `.env` с ключом Artemox.
2. Запустите `PlayGridAI.exe`.
3. Браузер откроется автоматически на `http://127.0.0.1:8080`.

## Что важно
- Ключ не вшивайте в код и не коммитьте в git.
- Если хотите отдать преподавателю один архив, положите в него:
  - `PlayGridAI.exe`
  - `.env`
  - `README_RUN.txt`

## Упрощённый сценарий для преподавателя
- открыть папку
- дважды нажать `PlayGridAI.exe`
- в браузере откроется сайт

## Формат сборки
Сборка настроена в формате one-file: получается один `PlayGridAI.exe`.
Файл `.env` читается автоматически из той же папки, где лежит `.exe`.
